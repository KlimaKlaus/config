# AGENTS.md — Nix Config Repo

This is a nix-darwin + home-manager configuration repo for macOS (Apple Silicon, `aarch64-darwin`). Every tool, dotfile, brew package, and macOS setting is declaratively managed here.

## Memory & Context

Session context is auto-injected by the vault hook (`~/.omp/agent/hooks/save-to-vault.ts`).
Recent session notes from `~/vault/projects/<org>/<repo>/<branch>/` are loaded at the
start of every session — no manual instruction needed.

## Architecture

```
flake.nix (entry) → darwin.lib.darwinSystem
  ├── nix/darwin.nix           imports darwin/* modules
  │   ├── darwin/system.nix    macOS defaults (dock, finder, trackpad)
  │   ├── darwin/services.nix  hostname, user, nix daemon config
  │   └── darwin/homebrew.nix  brew taps/brews/casks (opencode, ghostty, etc.)
  └── home-manager (integrated)
      └── nix/home.nix         imports home/* modules
          ├── home/packages.nix   CLI tools from nixpkgs (~50 packages)
          ├── home/shell.nix      zsh config, aliases, PATH, starship, shortcuts
          ├── home/git.nix        git config, gh, GPG, gpg-agent
          ├── home/tmux.nix       tmux binary
          └── home/dotfiles.nix   symlinks for dotfiles (starship, ghostty, tmux, etc.)
```

## Making changes

- **Add Nix package:** edit `nix/home/packages.nix` → `nrs`
- **Add Brew package:** edit `nix/darwin/homebrew.nix` → `nrs`
- **Change zsh config/aliases:** edit `nix/home/shell.nix` → `nrs`
- **Change dotfile:** edit the file (starship.toml, config.ghostty, etc.) → `nrs`
- **Quick alias test (no rebuild):** `echo 'alias ...' >> ~/.zshrc_local` → `exec zsh`
- **Change macOS setting:** edit `nix/darwin/system.nix` → `nrs`
- **Change git/GPG:** edit `nix/home/git.nix` → `nrs`

## What NOT to do

- **Never** `brew install` — brew is declarative via homebrew.nix
- **Never** edit `~/.zshrc` directly — it's generated from shell.nix, overwritten on rebuild
- **Never** commit `nix_secrets`, `.claude/.credentials.json`, `node_modules/`
- **Never** run `darwin-rebuild` without `sudo` after the initial bootstrap
- **Never** remove `nixpkgs.config.allowUnfree = true` from home.nix (needed for slack)

## Bootstrap (new machine)

```bash
# Install Nix, enable flakes, install Homebrew
git clone git@github.com:lucasfth/config.git ~/config
cd ~/config
touch nix_secrets
# Edit services.nix + flake.nix + home.nix for new hostname/username
nix build .#darwinConfigurations.<hostname>.system
./result/sw/bin/darwin-rebuild switch --flake .
sudo chsh -s ~/.nix-profile/bin/zsh $USER
```
