## Architecture

```
flake.nix (entry) → darwin.lib.darwinSystem
  ├── nix/darwin.nix              imports darwin/* modules
  │   ├── darwin/system.nix       macOS defaults (dock, finder, trackpad)
  │   ├── darwin/hostname.nix     hostname, user, nix daemon, GC
  │   ├── darwin/launchd.nix      launchd agents (borders, tailwind cleanup)
  │   ├── darwin/services.nix     Nix services stubs (postgres, redis)
  │   └── darwin/homebrew/        brew taps, brews, casks, mas, activation
  └── home-manager (integrated)
      └── nix/home.nix            imports common/* modules
          ├── common/packages/    CLI tools from nixpkgs (8 sub-modules)
          ├── common/shell/       zsh config split into init/aliases/paths/completions/env
          ├── common/git.nix      git config, gh, GPG, gpg-agent
          ├── common/tmux.nix     tmux binary
          ├── common/vim.nix      vim config + catppuccin theme
          └── common/dotfiles.nix symlinks for dotfiles (starship, ghostty, tmux, zed, etc.)
```

## Making changes

- **Add Nix package:** edit `nix/common/packages/<category>.nix` → `nrs`
- **Add Brew package:** edit `nix/darwin/homebrew/brews.nix` (formula) or `casks.nix` (cask) → `nrs`
- **Change zsh config/aliases:** edit `nix/common/shell/<file>.nix` → `nrs`
- **Change dotfile:** edit the file (starship.toml, config.ghostty, etc.) → `nrs`
- **Quick alias test (no rebuild):** `echo 'alias ...' >> ~/.zshrc_local` → `exec zsh`
- **Change macOS setting:** edit `nix/darwin/system.nix` → `nrs`
- **Change git/GPG:** edit `nix/common/git.nix` → `nrs`

## What NOT to do

- **Never** `brew install` — brew is declarative via nix/darwin/homebrew/
- **Never** edit `~/.zshrc` directly — it's generated from common/shell/, overwritten on rebuild
- **Never** commit `nix_secrets`, `.claude/.credentials.json`, `node_modules/`
- **Never** run `darwin-rebuild` without `sudo` after the initial bootstrap
- **Never** remove `nixpkgs.config.allowUnfree = true` from home.nix (needed for slack)

## Bootstrap (new machine)

```bash
# Install Nix, enable flakes, install Homebrew
git clone git@github.com:lucasfth/config.git ~/config
cd ~/config
touch nix_secrets
# Edit hostname.nix + flake.nix + home.nix for new hostname/username
nix build .#darwinConfigurations.<hostname>.system
./result/sw/bin/darwin-rebuild switch --flake .
sudo chsh -s ~/.nix-profile/bin/zsh $USER
```
